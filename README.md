## 継承したクラスを使う
継承したクラスを使うと「Bクラス」は「Aクラス」のすべての機能を引き継ぐことができる。
そのため、「Bクラス」内にはまだ何もメソッドは定義されていないが、「Aクラス」に定義されている「メソッド」などを使用することが可能になる。
